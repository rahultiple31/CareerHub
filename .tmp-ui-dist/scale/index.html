Heavy-scale architecture

Microservices, WebSocket events, search, recommendations, payments, and cloud storage.

Frontend

React or Next.js with TypeScript, role-aware routing, live data queries, SSR for public jobs, and CDN cached assets.

API Gateway

JWT auth, MFA, RBAC policy checks, rate limits, audit logs, and request routing into domain services.

Services

User, Job, Project, Interview, Video, Notification, Payment, Search, and Recommendation services.

Real Time

Socket.io or WebSocket gateway, Redis Pub/Sub for fanout, Kafka for durable interview, feed, and payment events.

Data

PostgreSQL for core transactions, MongoDB for feed and notes, Redis cache, object storage on S3 or Azure Blob.

Cloud Scale

Docker, Kubernetes, Helm, Terraform, autoscaling, multi-region read replicas, observability, and CI/CD pipelines.
